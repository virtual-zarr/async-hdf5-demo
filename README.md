# async-hdf5 demo

Benchmarking [async-hdf5](https://github.com/maxrjones/async-hdf5) + [zarrs-python](https://github.com/ilan-gold/zarrs-python) for reading ICESat-2 ATL06 from the cloud.

See the [full docs](./index.md) for details.

## Running locally

```bash
uv sync
uv run jupyter lab
```

Requires a [NASA Earthdata](https://urs.earthdata.nasa.gov/) account.
